Pi 400 Setup & Deploy

10:45 – 11:30
Before you start: This is your machine for the whole camp. One typo in a command can break everything — read twice, type once. Call your partner over if something looks wrong.
  1. 01
    Plug in your Pi 400 you — power cable, HDMI to monitor (use the port closest to the USB ports), ethernet into the switch. Give the ethernet 30–60 seconds to connect.
  2. 02
    First boot wizard you — set country, language, timezone. Username = your first name, password = Password01. Skip the software update prompt — your SD card is already updated.
  3. 03
    Open Terminal you — click the black monitor icon in the taskbar. Your prompt will look like yourname@raspberrypi:~ $ — that's normal.

Command reference

step 1 — install ollama (~2 min)
# Download and install Ollama
$ curl -fsSL https://ollama.com/install.sh | sh
Expected warning: "no NVIDIA/AMD GPU detected, running in CPU-only mode" — completely normal on a Pi 400. Ignore it and keep going.
step 2 — pull gemma 2b (~2 min)
# Download the Gemma 2B model to your Pi
$ ollama pull gemma:2b
step 3 — run your model
# Start Gemma 2B — wait ~10 seconds for the >>> prompt
$ ollama run gemma:2b
First prompt to try: "What is artificial intelligence in one sentence?" — watch it type one word at a time. That's the model predicting the next token, live. Expect ~35 seconds for a full answer — your Pi is doing this on a CPU the size of a stamp.
inside the >>> prompt
# Exit Ollama — ALWAYS /bye, not exit or quit
>>> /bye
# Clear conversation history
>>> /clear
# Show available commands
>>> /help
